### What's new?

**26/05/20**: 修复了一些堆积已久的问题，使注册机更隐蔽。

**详细更新日志**：

1. 允许不配置代理。在 .env 中设置空字符串即可。
2. 更新了 Request Body，使用 AES 加密密码等数据，适配 JavaScript 逻辑。
3. 更新了 Request Headers，新增 Referer 以适配 4399 新风控机制。
4. 使用 Faker 生成 User-Agent，无需硬编码配置，每次生成新的 User-Agent，使请求更隐蔽。

**但目前由于 4399 风控过于严格，无法保证可用。典型特征为：`网络错误_202`。**

### 项目特色

1. 唯一有自训练验证码模型的 4399 注册项目，正确率可达95%以上，保证稳定性

2. 优化最好的全自动注册项目，功能简洁干练，代码清晰易懂

3. 在搭配优秀的代理 IP 时，注册速度可达 7200 个/小时以上，保证账号供应速度（是指单线程，多线程会翻几十倍）

4. 具有可配置的多线程功能，注册速度大幅翻倍，每分钟注册账号数量可达上千个！

5. 唯一使用 Python 实现 4399 账号转 Cookie 功能的项目

6. 更好的 TUI，在终端下拥有良好的可视化

###  怎么用？

1. 将项目克隆到本地

2. 安装所有依赖

3. 修改 .env 配置

3. Enjoy :)

### 注意事项

1. 单 IP 每小时仅可注册 300 个，建议使用代理 IP。

2. 代理 IP 请自行解决。
